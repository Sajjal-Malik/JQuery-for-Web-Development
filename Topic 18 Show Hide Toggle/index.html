jQuery Show/Hide/Toggle Demo

Welcome to jQuery Effects!

This content can be shown, hidden, or toggled using jQuery methods.

Try clicking the buttons above to see the effects in action.

// Basic jQuery syntax for show/hide/toggle
                    $("#showBtn").click(function() {
                    $("#contentBox").show();
                });

                $("#hideBtn").click(function() {
                $("#contentBox").hide();
                });

                $("#toggleBtn").click(function() {
                $("#contentBox").toggle();
                });
